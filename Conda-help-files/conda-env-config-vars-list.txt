usage: conda-env-script.py config vars list [-h] [-n ENVIRONMENT | -p PATH]
                                            [--json] [-v] [-q]

List environment variables for a conda environment

Options:

optional arguments:
  -h, --help            Show this help message and exit.

Target Environment Specification:
  -n ENVIRONMENT, --name ENVIRONMENT
                        Name of environment.
  -p PATH, --prefix PATH
                        Full path to environment location (i.e. prefix).

Output, Prompt, and Flow Control Options:
  --json                Report all output as json. Suitable for using conda
                        programmatically.
  -v, --verbose         Use once for info, twice for debug, three times for
                        trace.
  -q, --quiet           Do not display progress bar.

examples:
    conda env config vars list -n my_env
