=================
Cmdlet Help Files
=================

The Posh Predictive Text module includes several PowerShell cmdlets. Each
cmdlet has its own help file accessed using ``Get-Help``. For example:

.. code-block:: powershell

    Get-Help Get-PredictiveTextOption
